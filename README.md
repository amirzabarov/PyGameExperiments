## Controls
Draw cells — left mouse button

Erase cells — lefts mouse button + space

Randomise cells — space + ctrl

Camera movement — arrow keys

Zoom in/out — W / S

Launch game of life — 1

Pause game of life — 2

Clear surface — 3
